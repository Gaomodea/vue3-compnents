
# 变量、作用域、上下文

```js
// 只有引用值可以动态添加后面可以使用的值
let name = "Alice";
name.age = 27;
console.log(name.age);  // undefined
```

```js
// 任意引用类型都是Object的实例
console.log(person instanceof Object);  // 变量person是Object吗？
console.log(colors instanceof Array);   // 变量colors是Array吗？
console.log(pattern instanceof RegExp); // 变量pattern是RegExp吗？

// instanceof对于原始类型都会返回false
'' instanceof String // false
```

### 作用域链

> - 全局上下文（window）、函数上下文（arguments）
> - 使用var声明变量时，变量会自动添加到最接近的上下文中；如果没有加var,就会被声明到全局上下文中，严格模式不支持
> - `var会被拿到函数或全局作用域的顶部，位于作用域的所有代码之前（变量提升）`
> - 全局作用域 > 顶级作用域 > 函数作用域 > with/catch作用域 > 块级作用域
