
# 数据结构

## Map

> - Map中的键可以是任意数据类型，Object只能是数字、字符串、Symbol
> - Map中几乎已严格相等来比较键的
> - Map中键的顺序是以插入的顺序来决定的，在迭代中保持相同的顺序

```js
let map = new Map([
  ['key', 'value'],
  [2, 2]
])
let map = new Map()

map.set(1, 1)
map.set('1', 2)

map.size // 2

let obj = {}
obj[1] = 1
obj['1'] = 2

obj.keys().length // 1
```

## 迭代器

> - 迭代：重复、循环
> - 可迭代对象：实现了Iterable(可迭代协议)接口，通过Iterator(迭代器)消费，具有有限的元素，无歧义的遍历顺序

::: warning 内置的可迭代对象和能遍历可迭代对象的结构
- 内置可迭代对象
  - Array
  - String
  - Map
  - Set
  - arguments
  - NodeList（DOM集合）
- 能接受可迭代对象的原生特性
  - for-of
  - 解构赋值
  - 扩展操作符
  - Array.from()
  - Map、Set的构造函数
  - `Promise.all、Promise.race`
  - `yield *`
- 通过 `Sybmbol.iterator`属性判断对象是否是可迭代对象
- Symbole.iterator属性可继承
:::

```js
// 可迭代对象：需要具有Symbol.iterator迭代器工厂方法，返回迭代器
class Counter {
  limit = 10

  // 实现可迭代接口
  [Symbol.iterator] () {
    let count = 0
    let limit = this.limit

    // 实现迭代器接口 (迭代器协议)
    return {
      next () {
        if (count < this.limit) {
          return { done: false, value: count++ }
        } else {
          return { done: true, value: undefined } // done: true是，value不会当作迭代的结果
        }
      },

      return() {
        console.log('迭代提前终止时调用')
      },

      // 迭代器实现可迭代接口，返回自身
      [Symbol.iterator] () {
        return this
      }
    }
  }
}

// 创建迭代器
const couter = new Counter()
const iter = counter[Symbol.iterator]()

// 遍历数据
for (const count of iter) {
  console.log(count) // 1, 2, 3, 4, 5

  if (count >= 5) {
    break; // 提前结束，调用iter.return()
  }
}

// 如果迭代器没有结束，可以从上一次离开的位置继续迭代
for (const count of iter) {
  console.log(count) // 6, 7, 8, 9
}

// 内置数据结构的迭代器本身也实现了Iterable，它返回的就是迭代器本身
const arr = [1, 2]
const iter = arr[Symbol.iterator]()
const iter2 = iter[Symbol.iterator]()
console.log(iter === iter2) // true

```

## 生成器

> - 生成器函数返回的生成器也是一个可迭代对象和迭代器
> - 生成器非常适合做递归，和函数递归有本质区别，很难触发栈溢出

```js
function * counter(n) {
  while(--n >= 0) {
    yield n
  }
}

for (const c of counter(5)) {
  console.log(c) // 4, 3, 2, 1, 0
}

class Counter {
  list = [1, 2, 3];

  // 通过生成器函数返回迭代器
  * [Symbol.iterator] () {
    try {
      yield* this.list
    } catch(e) {
      console.log('catch throw', e)
    }
  }
}

let cou = new Counter()
let iter = cou[Symbol.iterator]()
for (const v of iter) {
  if (v === 2) {
    iter.return('result') // 关闭生成器
  }
  console.log(v) // 1, 2

  // iter.throw('error')
}

// 生成器还有 throw方法
```
